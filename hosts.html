Hear from listeners

Listeners in your group can leave comments at any point in a podcast.

Supports both text and voice recordings.

Embed comments into the podcast

If there's a text comment @27:15, listeners will get a text notification when they reach that point in the show.

If it's a voice comment, the show will pause, play the voice comment, and then resume.

You can control which comments get played.

Learn more about embedded comments.

Add post-episode commentary

Want to mention something after publishing the episode? Drop a voice comment at any timestamp.

With embedded comments, it will automatically be added to the show!

Links in real time
