HNbuddy is built using several open source component. If you're not a Swift developer, don't worry, you can still help!

HNScraper

{{ gmc.gmc("tsucres/HNScraper") }}

HNScraper handles all the interactions with the website. This includes news fetching, favoriting/upvoting items, as well as logging into hacker news.

SwiftyMercuryReady

{{ gmc.gmc("tsucres/SwiftyMercuryReady") }}

This component is used to render the news with customizable font and theme.

SwiftyComments

{{ gmc.gmc("tsucres/SwiftyComments") }}

To display the discussion threads, HNBuddy uses the SwiftyComments module.

SwiftyComments

{{ gmc.gmc("tsucres/HeaderedTabScrollView") }}

The user page is built on top a HeaderedTabScrollView.
