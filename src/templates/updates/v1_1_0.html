v 1.1.0 - (In Review)

New themes

You can chose between several new themes and mark them as your preferred light/dark theme to replace the default ones.

More options

The "preferences" page was rebuilt from scratch to add several new options: expand discussions by default, disable the reader by default, ...

Fix bugs

Fixed several bugs on the "Lists" tab.

  • 2 new themes: Reader Dark and Reader Light. You can select your preferred light and dark themes. You can then manually switch between them or let the app pick the right one for you.
  • New list layout style: dense layout. You can choose the default style in the options or switch between them manually using the toggle switch on the top of the news list on the home page.
  • New contribution guide + contact form
  • New options:
    • Disable the automatic reader, open directly the webpage instead
    • Expand all the comments/replies by default
    • Disable the rearrangement of the comment section when a comment is expanded: by default the comment section is scrolled to make the first reply visible.
  • Bug fixes:
    • Cut text on the login form
    • Empty state placeholder that wasn't disappearing in the "Submissions" list
    • Fixed sort filters in the lists (they didn't work)
    • Load more than 30 bookmarks & recently read news (in the lists)
